# challenge-apple-devoloper-academy

você controla um mago iniciante que precisa enfrentar 6 inimigos, cada um mais forte que o anterior. No começo, você possui apenas:
1 magia de dano básico 
1 magia de cura básica 
A cada inimigo derrotado, você ganha uma nova magia para usar nas próximas batalhas. O objetivo é sobreviver aos quatro combates usando suas magias de forma estratégica.
O jogo é totalmente em texto, rodando no terminal, e funciona por turnos:
você escolhe uma magia → o inimigo ataca → próxima rodada. 
Se sua vida chegar a 0, o jogo acaba.
Se derrotar os 6 inimigos, você vence.

comandos
o jogo aceita apenas comandos numericos para as alternativas
números de magias que não existirem acabam por passar o seu turno, fazendo com que o inimigo ataque
